The <img> tag has two required attributes:
src - Specifies the path to the image
alt - Specifies an alternate text for the image

(21). How do you select a specific element within another element in CSS?

ANS - To select a specific element within another element in CSS, you can use a combination of selectors known as descendant selectors or child selectors.

Descendant Selector: To select an element that is a descendant (nested within) of another element, you can use a space between the selectors.
div span { /* CSS properties */ }

child Selector:To select an element that is a direct child of another element, you can use the > symbol between the selectors.
ul > li { /* CSS properties */ }

(29). How do you horizontally display a group of elements in CSS?

ANS -To horizontally display a group of elements in CSS, you can use the display property
with the value inline, inline-block, or flex. Each of these display values has different behavior
and usage depending on the specific requirements of your layout. inline: Set the display property to inline to make elements flow horizontally in a line,
similar to text. Here's an example: .element { display: inline; } inline-Block: Set the display property to inline-block to make elements flow horizontally
like inline elements, but with the ability to set width, height, and margins. Here's an example: .element { display: inline-block; } Flexbox: Set the display property to flex on the container element and apply appropriate flexbox
properties to the child elements to create a flexible layout with horizontal display. Flexbox provides
powerful features for aligning and distributing elements within a container. Here's an example: .container { display: flex; } <div class="container"> <div class="element">Element 1</div> <div class="element">Element 2</div> <div class="element">Element 3</div> <div class="element">Element 4</div> </div>

(30). How do you apply a hover effect to an element in CSS?

ANS - To apply a hover effect to an element in CSS, you can use the :hover pseudo-class. The :hover pseudo-class targets an element when it is being hovered over by the cursor. You can then apply specific styles to the element when the hover event occurs.

selector:hover {
/* styles applied when the element is being hovered over */
}
